# 3D Assignment

Mini web aplikacija za rad sa dva GLB modela uz Next.js, Three.js i Firestore.

## Funkcionalnosti

- ucitavanje dva GLB modela
- drag and drop modela po sceni
- prebacivanje izmedju 3D i top-down 2D pogleda
- edit rotacije modela
- cuvanje pozicije i rotacije u Firestore

## Instalacija
- npm install
- npm run dev
- Otvoriti na http://localhost:3000
